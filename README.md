# dsa-playground

1. เขียนเล่นตามหนังสือ
2. เรียนรู้ C++
3. ฝึกเขียน test


**Setup**
```bash
cmake --preset default
```

**Build**
```bash
./build <filename> 
./build [--all|-a]
./build EuclidsAlgo
```

**Run**
```bash
./run <filename> [args...]
./run HelloWorld "Thas"
```

**Test**
```bash
./test <testregex>
./test [--all|-a]
./test [--list|-l] <testregex>
```
